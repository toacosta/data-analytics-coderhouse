# Data Analytics - Coderhouse

## 📖 Descripción

Este repositorio contiene los ejercicios, proyectos y prácticas desarrollados durante mi formación en Data Analytics en Coderhouse.

El objetivo de este repositorio es documentar mi aprendizaje y construir un portfolio con proyectos aplicados de análisis de datos.

---

## 🛠 Herramientas utilizadas

- SQL Server
- SQL Server Management Studio (SSMS)
- Microsoft Excel
- Power BI
- Git & GitHub

## 📂 Estructura del repositorio

```text
01_SQL
02_Excel

05_Datasets
```

Cada carpeta contiene los ejercicios, proyectos y entregas correspondientes a cada módulo del curso.

## 🎯 Objetivo

Aplicar técnicas de análisis de datos para resolver problemas de negocio mediante el uso de SQL, Excel, Power BI y otras herramientas de análisis.
